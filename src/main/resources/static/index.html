DESCUBRA! API

Classe Administrators

  • /administrators
    @GET
    RETORNA TODOS OS ADMINISTRADORES DO SERVIÇO.

  • /administrators/{admId}
    @GET
    PEGA O ADMINISTRADOR PELO ID.

  • /administrators
    @POST
    CRIA O ADMINISTRADOR.

  • /administrators/{admId}
    @DELETE
    APAGA O ADMINISTRADOR PELO ID.

  • /administrators/{admId}/events
    @GET
    PEGA TODOS OS EVENTOS DAQUELE ADMINISTRADOR PELO ID.

  • /administrators/{admId}/events
    @POST
    CRIA OS EVENTOS DAQUELE ADMINISTRADOR PELO ID.
